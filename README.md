# Hotel Performance Dashboard

A data engineering project focused on building an end-to-end analytics pipeline. This dashboard visualizes hotel booking performance, revenue trends, and key KPIs using a cloud-based data stack.

## Project Overview
This project demonstrates the end-to-end data lifecycle:
* **Data Storage & Processing:** Snowflake & Snowpark for efficient data cleaning and transformation.
* **Dashboarding:** Streamlit for building a clean, interactive user interface.
* **Visualization:** Altair for generating performance metrics and trend analysis.

## Key Features
* Real-time revenue tracking and trend analysis.
* Geographic booking performance (Top Cities).
* Interactive data grid for detailed record viewing.
